# Online-Slicer 🖨️

**Онлайн-платформа для просмотра и слайсирования 3D-моделей для 3D-печати**

Полнофункциональное веб-приложение для работы с 3D-файлами формата STL и других форматов печати. Позволяет загружать модели, просматривать их в реальном времени и использовать встроенный слайсер для подготовки к печати.

## 🌟 Особенности

- **Просмотр 3D-моделей** — поддержка STL и других форматов файлов печати
- **Интерактивная 3D-сцена** — масштабирование, вращение, перемещение моделей
- **Встроенный слайсер** — подготовка моделей к печати (на основе Kimi Moto)
- **Отзывчивый интерфейс** — оптимизирован для десктопа и мобильных устройств
- **Реальное время** — облачная обработка и хранение моделей
- **API для интеграции** — REST API для внешних приложений

## 🚀 Быстрый старт

### Требования
- **Node.js** 18.x или выше
- **npm** 9.x или выше
- **MongoDB** (для хранения данных)

### Установка

**1. Клонируй репозиторий:**
```bash
git clone https://github.com/Mukller/Online-slicer.git
cd Online-slicer
```

**2. Установи зависимости для бэкенда:**
```bash
cd backend
npm install
```

**3. Установи зависимости для фронтенда:**
```bash
cd ../frontend
npm install
```

**4. Установи зависимости для приложения печати:**
```bash
cd ../print3d-app
npm install
cd ..
```

### Запуск

**Запуск бэкенда:**
```bash
cd backend
npm start
```
Сервер запустится на `http://localhost:5000`

**Запуск фронтенда (в другом терминале):**
```bash
cd frontend
npm run dev
```
Приложение откроется на `http://localhost:5173`

**Запуск приложения печати (в третьем терминале):**
```bash
cd print3d-app
npm run dev
```

## 📚 Технологический стек

### Бэкенд
- **Express.js** — веб-фреймворк
- **Mongoose** — ODM для MongoDB
- **Multer** — обработка загрузок файлов
- **CORS** — кросс-доменные запросы

### Фронтенд
- **React 19** — UI библиотека
- **Vite** — сборщик проекта
- **Three.js** — 3D графика
- **React Three Fiber** — React интеграция для Three.js
- **Framer Motion** — анимации
- **Tailwind CSS** — утилиты CSS
- **Axios** — HTTP клиент

### БД и Хранилище
- **MongoDB** — NoSQL база данных
- **GridFS** — хранилище больших файлов (опционально)

## 🔌 API Endpoints

### 3D Модели
- `GET /api/models` — список всех моделей
- `POST /api/models/upload` — загрузка новой модели
- `GET /api/models/:id` — получить модель по ID
- `DELETE /api/models/:id` — удалить модель

### Слайсирование
- `POST /api/slice` — слайсировать модель
- `GET /api/slice/:jobId` — статус задачи слайсирования

### Пользователи (планируется)
- `POST /api/users/register` — регистрация
- `POST /api/users/login` — авторизация
