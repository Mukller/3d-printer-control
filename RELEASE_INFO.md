# Release Info

## Current Version: 0.1.0

**Released:** 2026-07-10  
**Type:** Initial release  
**Status:** Development

## Summary

Initial commit establishing the repository structure for the 3D printer control system. Contains configuration templates, setup scripts, and documentation based on a working OctoPrint + Klipper + Telegram bot setup for a Creality Ender 5 S1 printer.

## Contents

- OctoPrint plugin list and config templates
- Klipper `printer.cfg` template for Ender 5 S1
- Telegram bot (OctoTelegram) configuration template
- Setup automation scripts
- Full documentation for setup and plugins
- Telegram chat archive showing bot functionality (Sep–Oct 2024)

## Roadmap

- v0.2.0: Full Klipper calibration configs + macros
- v0.3.0: Auto-install script tested end-to-end on Pi
- v1.0.0: Ready-to-flash Pi image with everything pre-configured
